## How to Use

### With NanoBanana (Canva MCP)
Use the full prompt as-is. Set dimensions to 1200x675 or 1920x1080.

### With Sora
Append to any prompt: `camera slowly orbiting around the scene, volumetric light rays shifting, caustic patterns moving gently, 4 seconds, cinematic`

### With FLUX / Stable Diffusion
Use the prompt as positive prompt. Negative prompt:
```
text, words, letters, watermark, signature, low quality, blurry, flat, matte, opaque, rough texture, fingerprints, scratches, dirty, noisy, cartoon, anime, painting, illustration, human face, realistic skin
```

### After generating
1. Save as WebP (quality 85) to `public/images/blog/<slug>.webp`
2. Add `image: /images/blog/<slug>.webp` to the post frontmatter or directly in `blogData.ts`
3. Run `npm run publish` or manually update
